人稱代名詞 pronombre personal

José自己覺得西班牙文最需要耗費心思理解的部分，非代名詞用法莫屬了。相信大家在不論是課本還是影集中，都會看到如 el, lo/los, la/las, le/les, se, me, te, os, nos 這些字詞出現。雖然它們都只有短短兩到三個字母，但是針對它們應該出現的位置、我們該如何解讀，其實背後都有很大的學問！事不宜遲，我們先來個概念性的介紹吧。

主格、受格、所有格

主格
sujeto		 受格
complemento		 所有格
posesivo
我 yo a mí mi, mis
你 tú a ti tu, tus
他/她/您 él / ella / usted a + él / ella / usted su, sus
我們 nosotros/as a nosotros/as nuestro/a/s
你們 vosotros/as a vosotros/as vuestro/a/s
他們/她們/您們 ellos / ellas / ustedes a + ellos / ellas / ustedes su, sus

這裡有幾個重點要跟大家提醒：

  1. 主格的你和他，分別是有重音的 tú 和 él。如果沒有重音的話，tu 是所有格「你的」的意思；el 則變成陽性單數的定冠詞。
  2. 如果在我們、你們、他們當中有一位男性的存在，主格就使用陽性的 nosotros、vosotros、ellos；如果全為女性，則使用陰性的 nosotras、vosotras、ellas。
  3. 受格前面的 a 是介係詞，在西班牙文中，如果某個動詞的受詞是人的話，在動詞和受詞之間要加上 a，por ejemplo：Os compro dos entradas de concierto a vosotros. 我買了兩張演唱會的票給你們。補充：有些動詞有習慣搭配的介係詞，此時不一定會用到 a 加上受格，可能是 en + 受格等等。
  4. 在受格的部分，請特別注意介係詞後第一、二人稱單數型的樣貌，以英文為例，我們會說 for me / you / her...，對應到西文則是 para mí / ti / ella。
    • 其他人稱遇上任何介係詞，其受詞的形貌等同於主格，如：a ellos, sin vosotros, con ella。
    • 但如果受詞是我、你，則會是 a + mí / ti, para + mí / ti, sin + mí / ti ...。唯一的例外是搭配介係詞 con，此時會長成 conmigo / contigo。
    • 特別注意，如果寫成沒有重音的 mi，是所有格「我的」的意思。
  5. 在所有格的部分大家會發現有兩種以上的樣子存在，mi 跟 mis，vuestro 跟 vuestras 等等，這是因為所有格要隨著修飾的名詞做單複數、陰陽性的變化。

小提醒：您/您們的直接受詞應該用 lo/los 還是 la/las 取決於對象的性別；另外大家常見的「me gusta」句型，在提到「他喜歡」時，記得是「le gusta」而非「se gusta」喔！José的西班牙文老師講到這裡都會很激動，說要把學了兩年西文卻還是講 se gusta的人當掉 XD。其他的細節由於內容過多，需要另外一篇文章跟大家解釋，因此在這裡只初步的跟大家介紹。

小提醒：所有格代名詞在英文裡就是 mine / yours...，以「這是我的書」為例：Esto es mi libro. = Este libro es mío.。José 個人覺得這是比較難用中文解釋的概念，不過我會盡量的試著說明清楚！請參考下面所列出的文章囉。

以上，就是最初步的介紹啦，希望大家看到這邊之後，就不會那麼容易的被代名詞搞得暈頭轉向。可是由於資訊量實在過於龐大，José認為需要再細講的部分就放到其他頁面囉，詳細的內容就請點選下方的選單：

  • 直接受詞、間接受詞：該怎麼用？要放在哪？
  • 所有格、所有格代名詞：該怎麼用？要怎麼變化？
  • 反身動詞：使用的時機？

另外，代名詞除了代指「人」以外，還可以代稱「物」，強力推薦大家看這支youtube影片，內容相當詳盡跟清楚，相信對大家學習西文會很有幫助＞＜
